# [Programmers 43105 - 정수 삼각형](https://school.programmers.co.kr/learn/courses/30/lessons/43105)

## 문제 설명

![](./assets/photo1.png)

위와 같은 삼각형의 꼭대기에서 바닥까지 이어지는 경로 중, 거쳐간 숫자의 합이 가장 큰 경우를 찾아보려고 합니다. 아래 칸으로 이동할 때는 대각선 방향으로 한 칸 오른쪽 또는 왼쪽으로만 이동 가능합니다. 예를 들어 3에서는 그 아래칸의 8 또는 1로만 이동이 가능합니다.

삼각형의 정보가 담긴 배열 triangle이 매개변수로 주어질 때, 거쳐간 숫자의 최댓값을 return 하도록 solution 함수를 완성하세요.

## 제한 사항

- 삼각형의 높이는 1 이상 500 이하입니다.
- 삼각형을 이루고 있는 숫자는 0 이상 9,999 이하의 정수입니다.

## 입출력 예

| triangle                                                | result |
| ------------------------------------------------------- | ------ |
| [[7], [3, 8], [8, 1, 0], [2, 7, 4, 4], [4, 5, 2, 6, 5]] | 30     |

## 코드

```java
class Solution {
    public int solution(int[][] triangle) {
        int len = triangle.length;

        int[][] dp = new int[1 + len][1 + len];
        for (int i = 1; i <= len; i++) {
            for (int j = 1; j <= i; j++) {
                dp[i][j] = Math.max(dp[i - 1][j - 1], dp[i - 1][j]) + triangle[i - 1][j - 1];
            }
        }

        int answer = 0;
        for (int n : dp[len]) {
            answer = Math.max(answer, n);
        }


        return answer;
    }
}
```

## 해설

다이나믹 프로그래밍을 활용하면 된다.

2차원 dp 배열은 삼각형에서 해당 위치까지 가장 큰 숫자를 통해 온 경우 숫자의 합을 저장했다. 이렇게 할 경우 마지막 행에서 가장 큰 숫자가 거쳐간 숫자의 최댓값이 된다. 각 칸의 경우 윗 칸과 윗 칸의 왼쪽 칸 중 더 큰 수에서 거쳐오는게 더 합이 커지고 이것이 반복되므로 반복문으로 쭉 돌면 되며 편의를 위해 가장 윗 행과 오른쪽에 패딩을 주었다.
