- 구현 시간 : 27min

# 💡 아이디어

- 처음에는 Queue를 이용해서 문제를 해결하려고 함
  - String[] 로 좌석 정보를 받고, 컵홀더가 들어갈 수 있는 자리는 -1, 좌석이 있는 자리는 1을 넣으려고 함
  - 이후 Queue를 돌며 2개의 원소의 합이 0이 될 때마다 카운트를 하려고 함
  - 근데 조건문이 까다로워질거 같아서 하다가 접음
- 컵홀더 배치를 보다가 고등학교 조합이랑 비슷한거 같아서 시도해봄(설명은 아래)
