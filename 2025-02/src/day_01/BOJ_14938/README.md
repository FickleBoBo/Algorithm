# 💡 아이디어

- 최단거리를 통해 아이템의 합을 구하는 문제로 다익스트라 알고리즘과 N이 최대 100이어서 플로이드 워셜 알고리즘으로 해결할 수 있다.

---

# ✔ 문제 풀이

- 다익스트라 알고리즘은 거리 배열을 반환 후 M 이하인 지역의 아이템을 줍는 방식으로 구현했고 플로이드 워셜은 dp 테이블로 바로 계산했다.
